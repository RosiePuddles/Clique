\section{Results}
This appendix contains the results from testing the Python implementation of the algorithm given in listing \ref{lst:pyo}. This only contains times. The algorithm gave the expected answer for all tested graph. The code for running these benchmarks along with the code for checking the algorithm and raw data in CSV format gives the expected value are available in this \href{https://github.com/RosiePuddles/Clique}{repository} in the \texttt{benchmark.py} and \texttt{test.py} files respectively
